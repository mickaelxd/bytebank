## Instalação

<div id="installation"></div>

**Você já deve ter instalado:**

-   [Flutter](https://flutter.dev)

**Vamos dividir em 3 etapas:**

1. Clone o repositório
2. Instale as dependências
3. Rode o projeto

### 1. Clone o repositório

```
git clone https://github.com/mickaelxd/bytebank.git
```

---

### 2. Instale as dependências

```
flutter pub get
```

### 3. Rode o projeto

```
flutter run
```
